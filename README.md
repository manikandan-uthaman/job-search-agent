# Nordic Job Agent

Agentic AI platform that helps discover software engineering opportunities in Sweden and Nordic countries.

## Goal

Build a multi-agent AI system capable of:

- Discovering recruiters and hiring managers
- Researching companies
- Ranking contacts
- Generating personalized outreach
- Tracking job search activities

## Technology Stack

- Java 21
- Spring Boot
- Spring AI
- LangChain4j
- PostgreSQL
- pgvector
- React
- Docker
- Kubernetes
